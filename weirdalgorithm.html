Visit WeirdAlgorithm.com

A Python application employing the Flask framework. The user enters song lyrics as well as words that he/she wishes to rhyme. The application parses the input and compares the words against a custom-built system of dictionaries built from the Carnegie-Mellon University rhyming dictionary. Rhyme words are swapped and the full lyrics are returned to the user.
